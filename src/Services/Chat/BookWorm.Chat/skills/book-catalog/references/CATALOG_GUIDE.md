# Book Catalog Guide

## Genre Descriptions

| Genre                | Description                    | Popular with               |
| -------------------- | ------------------------------ | -------------------------- |
| Fiction              | Narrative prose stories        | General readers            |
| Science Fiction      | Speculative future/technology  | Tech & sci enthusiasts     |
| Fantasy              | Magic and mythical worlds      | Adventure seekers          |
| Mystery / Thriller   | Crime, suspense, investigation | Plot-driven readers        |
| Non-Fiction          | Factual accounts and analysis  | Learners                   |
| Self-Help            | Personal development           | Goal-oriented readers      |
| Biography            | Life stories of real people    | History & inspiration fans |
| Children's           | Age-appropriate stories        | Parents, educators         |
| Young Adult (YA)     | Coming-of-age narratives       | Teens, adults              |
| Software Engineering | Programming and architecture   | Developers                 |
| Business             | Strategy, management, finance  | Professionals              |

## Search Tips

- **Broad first**: Start with genre or theme, then narrow by author or rating.
- **Author variations**: Try both full name and last name only (e.g., "Fowler" vs "Martin Fowler").
- **Series**: Mention the series name if asking about a sequence (e.g., "Dune series").
- **Gift searches**: Ask about the recipient's age, interests, and budget.

## Top-Rated Categories

1. Software Engineering — consistently high ratings from professional readers.
2. Science Fiction — broad appeal and deep catalog.
3. Mystery / Thriller — highest re-read rate.
